## [0.3.0] - 2026-01-04

### Added

- Streaming support via `CreateChatCompletionStream` using Bedrock's ConverseStream API
- `stream.go` with event handling for text deltas, tool use deltas, and metadata
- Tool/function calling support with `ToolConfiguration` mapping
- Tool choice options: auto, required, none, specific tool
- Streaming tool call support with incremental argument delivery
- Bearer token authentication via `AWS_BEARER_TOKEN_BEDROCK` environment variable
- Integration tests (8 tests) covering completions, streaming, and tool calling
- Unit tests (18 tests) with mock Bedrock client
- CI/CD documentation for GitHub Actions with both auth methods

### Changed

- README.md updated with credential chain priority table
- README.md updated with testing documentation and environment variables
- Bedrock model access section simplified (AWS auto-enables models on first invocation)

### Removed

- `types.go` removed (types consolidated into other files)

## [0.2.0] - 2025-12-15

### Added

- Core chat completion support via Bedrock Converse API
- System message handling (extracted to Bedrock system blocks)
- Multi-turn conversation support
- Inference configuration (MaxTokens, Temperature, TopP, Stop)
- Token usage reporting

## [0.1.0] - 2025-12-01

### Added

- Initial release
- Basic Bedrock client wrapper
- Provider interface implementation for omnillm
